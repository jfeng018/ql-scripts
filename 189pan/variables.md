# 天翼云盘自动签到脚本变量说明

## 需要配置的环境变量

在青龙面板中添加以下环境变量：

| 变量名 | 类型 | 必填 | 说明 | 示例 |
|--------|------|------|------|------|
| `TYYP_USERNAME` | String | 是 | 天翼云盘账号手机号，多个账号用 `&` 分隔 | `13800138000` 或 `13800138000&13900139000` |
| `TYYP_PSW` | String | 是 | 天翼云盘账号密码，多个账号用 `&` 分隔，与账号顺序一一对应 | `your_password` 或 `password1&password2` |

## 配置示例

### 单账户配置
```
TYYP_USERNAME=13800138000
TYYP_PSW=your_password
```

### 多账户配置
```
TYYP_USERNAME=13800138000&13900139000
TYYP_PSW=password1&password2
```

## 注意事项

1. 账号和密码的数量必须一致
2. 多账户之间使用 `&` 符号分隔
3. 环境变量名称必须完全一致，区分大小写
4. 建议定期检查账号密码是否有效
5. 不要在代码中硬编码账号密码，应始终使用环境变量
