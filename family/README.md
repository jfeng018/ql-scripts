# 家庭纪念日提醒脚本（合并变量版）

## 功能介绍

这是一个为青龙面板设计的家庭纪念日提醒脚本，可以：

1. 计算恋爱天数、结婚纪念日倒计时
2. 计算家庭成员生日倒计时
3. 计算孩子年龄
4. 通过青龙面板通知机制发送每日温馨提醒

## 环境变量配置

在青龙面板中配置以下环境变量：

### 推荐配置方式（合并为单个变量）：

| 变量名 | 示例值 | 说明 |
|--------|--------|------|
| FAMILY_DATES | 2020-01-01\|1990-05-15\|1988-12-03\|2021-10-01\|2023-03-20 | 恋爱日期\|妻子生日\|丈夫生日\|结婚纪念日\|孩子生日 |

格式说明：
- 使用竖线 `|` 分隔各个日期
- 顺序为：恋爱日期|妻子生日|丈夫生日|结婚纪念日|孩子生日
- 所有日期格式均为 YYYY-MM-DD

### 传统配置方式（兼容旧版本）：

| 变量名 | 示例值 | 说明 |
|--------|--------|------|
| FAMILY_LOVE_DATE | 2020-01-01 | 恋爱日期 |
| FAMILY_BIRTHDAY_WIFE | 1990-05-15 | 妻子生日 |
| FAMILY_BIRTHDAY_HUSBAND | 1988-12-03 | 丈夫生日 |
| FAMILY_BIRTHDAY_MARRY | 2021-10-01 | 结婚纪念日 |
| FAMILY_CHILD_BIRTHDAY | 2023-03-20 | 孩子生日 |

注意：如果配置了 FAMILY_DATES，则优先使用该变量，忽略其他单独的变量。

## 依赖安装

脚本会自动安装所需依赖，无需手动安装。

## 使用方法

1. 在青龙面板中配置环境变量
2. 将脚本添加到定时任务中
3. 设置合适的执行时间（建议每天早上8点）

## 通知方式

脚本支持青龙面板的所有通知方式，包括：
- Server酱
- Telegram
- 企业微信
- 钉钉
- 邮件
- 等其他青龙面板支持的通知方式

## 注意事项

1. 所有日期格式均为YYYY-MM-DD
2. 脚本会自动处理日期计算和倒计时
3. 无需额外的API密钥，纯本地计算
4. 如果某个日期不需要，可以留空或填写0000-00-00
