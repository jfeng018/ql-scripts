# 环境变量配置说明

## 必需环境变量

| 变量名 | 示例值 | 说明 | 是否必填 |
|--------|--------|------|---------|
| FAMILY_LOVE_DATE | 2020-01-01 | 恋爱日期 | 是 |
| FAMILY_BIRTHDAY_WIFE | 1990-05-15 | 妻子生日 | 是 |
| FAMILY_BIRTHDAY_HUSBAND | 1988-12-03 | 丈夫生日 | 是 |
| FAMILY_BIRTHDAY_MARRY | 2021-10-01 | 结婚纪念日 | 是 |
| FAMILY_CHILD_BIRTHDAY | 2023-03-20 | 孩子生日 | 是 |

## 配置说明

### 日期格式
所有日期变量均使用 `YYYY-MM-DD` 格式，例如：`2020-01-01`

## 通知配置
脚本使用青龙面板内置的通知机制，支持多种通知方式：
- Server酱
- Telegram
- 企业微信
- 钉钉
- 邮件
- 等其他青龙面板支持的通知方式

请在青龙面板的系统配置中设置相应的通知参数。
