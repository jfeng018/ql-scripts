# 青龙面板通知模块测试脚本

## 功能说明

这是一个专门用于测试青龙面板通知模块是否正常工作的脚本。它可以：

1. 检查 `sendNotify.py` 文件是否存在
2. 尝试多种方式导入通知模块
3. 发送测试通知消息
4. 提供详细的调试信息

## 使用方法

1. 确保脚本已通过青龙面板的订阅功能自动更新到本地
2. 在青龙面板中找到 `notification_test/test_notification.py` 脚本
3. 手动运行该脚本
4. 查看运行日志，观察通知模块的导入和发送情况

## 测试内容

脚本会依次尝试以下几种导入方式：

1. `from sendNotify import send`
2. `import sendNotify`
3. 添加青龙面板特定路径后再导入

## 预期结果

- 如果通知模块正常工作，您应该会收到一条测试通知消息
- 如果通知模块有问题，脚本会在日志中显示详细的错误信息

## 故障排查

如果未收到通知，请检查：

1. 青龙面板的通知配置是否正确
2. 是否已配置有效的通知方式（如Server酱、Telegram等）
3. 相关环境变量是否已正确设置
